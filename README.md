# leandrocp dotfiles

* bash
* zsh
* sqlite
* git
* rbenv

Dependencies
------------
  
* [ZSH](http://www.zsh.org/)

Installation
------------

    sudo apt-get install git-core ruby rake
    git clone --recursive git@github.com:leandrocp/dotfiles.git ~/.dotfiles
    cd ~/.dotfiles
    rake install

Optional
--------

* [brew](https://github.com/mxcl/homebrew)
* [Pry](http://pry.github.com/)
